Set kernel fuction. kOption is s string.

"box"-> take average of several bins.

Rebin histograms befor smoothing.
-1 means dont rebin. WARNING: if you set this and use it in a loop over systematic hists you may rebin nominal histogram several times.

void SmoothHist::setStatErrThreshold ( float  ErrFraction  )  [inline]

This applys to smoothing with kernel.
Set threshold for stat error.
if stat error of bins are larger then this threshold merge bins between two extrema untill number of extrema reach the value set by setNmax(N).
If total stat error of nominal hist larger then the threshold, skip smoothing will appy.
